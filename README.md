# API REST - Baozi Store

Projeto desenvolvido para a disciplina **Desenvolvimento Web Back-End**.

## Tecnologias utilizadas

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Descrição

A Baozi Store é uma pequena loja fictícia que vende pãozinho chinês. Esta API REST foi desenvolvida para realizar o gerenciamento de:

- Clientes
- Produtos
- Pedidos

Cada pedido registra:

- Cliente
- Produto
- Quantidade

## Endpoints implementados

### Cliente
- POST /clientes
- GET /clientes
- GET /clientes/{id}
- DELETE /clientes/{id}

### Produto
- POST /produtos
- GET /produtos
- GET /produtos/{id}
- DELETE /produtos/{id}

### Pedido
- POST /pedidos
- GET /pedidos
- GET /pedidos/{id}
- DELETE /pedidos/{id}
